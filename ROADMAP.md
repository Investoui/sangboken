# Sangboken - Roadmap

## Strategi: Freemium + Utvidet bibliotek

### Fase 1: Rebranding (ferdig)
- [x] Oppdater slogan til "Den digitale sangboken med gitarakkorder"
- [x] Fjern "barnesanger" fra hovedtekst – bruk "norske sanger"
- [x] Oppdater meta/SEO med ny posisjonering
- [x] Oppdater CLAUDE.md med ny branding

### Fase 2: Utvid innhold (1-3 mnd)
- [ ] Legg til voksensanger – start med allsangklassikere:
  - "Vår beste dag" (Plumbo)
  - "En vill og en rose" (Lillebjørn Nilsen)
  - "Sommerfuggel i vinterland" (Hellbillies)
  - Tradisjonelle viser (Prøysen, Alf Prøysen)
- [ ] Kategoriser sanger: Barn, Allsang, Viser, Sesong
- [ ] Merk premium-innhold (men hold tilgjengelig midlertidig)

### Fase 3: Lansér Premium (3-6 mnd)

**Gratis tier:**
- Alle barnesanger (eksisterende bibliotek)
- Grunnleggende visning
- Akkorddiagrammer
- Transponering

**Premium tier (299-399 kr/år):**
- Fullt sangbibliotek (voksensanger, viser)
- Setlister / spillelister
- Offline-modus (PWA)
- Karaoke-modus (kun tekst)
- PDF-eksport
- Prioritert support

### Fase 4: Vekst (6-12 mnd)
- [ ] B2B-lansering for institusjoner (barnehager, skoler, eldrehjem)
- [ ] Markedsføring via familieblogger, musikklærere
- [ ] Utvidelse til dansk/svensk marked?

---

## Konkurransefordeler

1. **Norsk fokus** - Ingen andre tar det norske markedet seriøst
2. **Fungerer overalt** - Mobil, nettbrett, storskjerm (AirPlay/Chromecast)
3. **Gratis og reklamefri** - Ingen distraksjoner
4. **Kulturell resonans** - "Den digitale sangboken" spiller på tradisjon
